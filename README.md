# 📌 Job Application Tracker System

A backend system that helps users track and manage their job applications efficiently.  
It is also a personal productivity tool built to help organize job search progress while showcasing backend and DevOps skills.

---

## 🚀 Features

### 👤 Authentication
- User registration
- Secure login using JWT
- Protected routes

### 💼 Job Applications Management
- Create job applications
- Update application details and status
- Delete applications
- View all applications per user

### 📊 Tracking System
- Track application status:
  - Applied
  - Interview
  - Rejected
  - Offer
- Add notes per application
- Store job links and company details

### 🔍 Filtering & Insights
- Filter by status
- Sort by date applied
- Basic dashboard statistics:
  - Total applications
  - Interviews
  - Rejections
  - Offers

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- PostgreSQL (Railway hosted)
- Railway (Deployment + Database hosting)
- JWT Authentication
- Docker
- GitHub Actions (CI/CD)
